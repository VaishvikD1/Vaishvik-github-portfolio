Overall Formatting

The overall site colors are Red (#cc0000) and Black (#000000).

The general font-family is Times New Roman with Julius Sans One as a fallback.

The general text size is 15pt with a line-height of 1.15.

Site Header / Branding

The header uses the font Franklin Gothic Medium (fallback: Arial Narrow).

Header background is red (#cc0000) with white (#ffffff) text for contrast.

CSS classes applied: .site-header and .site-title

Navigation Links

Navigation uses Franklin Gothic Medium font, matching the header.

Nav bar background is dark black (#1a1a1a) with white link text.

On hover, links get a red (#cc0000) background with rounded corners.

CSS classes applied: .nav-bar and .nav-link

Content Section

Content uses a max-width of 900px, centered with auto margins.

Padding is 25px top/bottom and 30px left/right.

Background color is dark (#1a1a1a) with a subtle border-radius of 6px.

CSS classes applied: .content-section and .body-text

Site Footer

Footer background color is dark black (#1a1a1a).

Text is white (#ffffff), centered, with a line-height of 1.15.

CSS classes applied: .site-footer

Stylesheet Types Used

External stylesheet: _css/format.css — contains 10+ CSS rules using class selectors.

Embedded styles: defined in a <style> tag in the <head> — sets body background, link color, and h2 font.

Inline styles: applied directly to elements — e.g. red color on <h1>, font-size and line-height on <p>, and max-width on <img>.
