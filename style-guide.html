Colors

Background: #1a1a2e (dark navy) for header and footer

Accent: #00d4ff (cyan) for links and hover states

Body background: #f4f4f4 (light gray)

Content background: #ffffff (white)

Typography

Font family: Arial, sans-serif

Body font size: 1.1em with 1.7 line height for readability

Headings: bold, sized with em units

Layout

Pages use four structural sections: header, nav, main, and footer.

Main content is centered with a max-width of 800px and auto margins.

CSS Selectors Used

Descendant selectors: header h1, nav ul, nav a, main p, main img, footer a

Pseudo-class: nav a:hover, footer a:hover
